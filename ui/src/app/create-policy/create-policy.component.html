Escalation rule

Escalation rule consists of levels, each of which you will add users to and set up a timeout for. Whenever there is an incident in a service, the users will be notified and after the timeout, this rule will escalate to the next levels if no one acknowledges the incident.

Drag the numbers to rearrange the levels.
